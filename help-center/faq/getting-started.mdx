## Getting Started

To create your first documentation project:

1. Go to `https://docs.anilkumar.com`
2. Sign in with your account
3. Click **New Project**
4. Enter a project name (e.g., "My Docs")
5. Start adding pages and sections

## Edit Your Docs

- Click a page title to begin editing
- Use Markdown for formatting

Your changes save automatically!
