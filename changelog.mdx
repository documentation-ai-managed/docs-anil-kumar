## Upgrade Process

Follow these steps to upgrade your Anil Kumar documentation instance:

<Steps>
  <Step title="Backup" icon="database">
    Export your current site configuration.

    ```bash
    npx anilkumar-docs export --backup
    ```
  </Step>
  <Step title="Update" icon="download">
    Run the upgrade command with the latest version.

    ```bash
    npx anilkumar-docs upgrade v1.2.0
    ```
  </Step>
  <Step title="Verify" icon="check-circle">
    Test your pages for parsing errors.

    <Callout kind="tip">
      Scan for unescaped `{` or `<` characters.
    </Callout>
  </Step>
</Steps>
