## Overview

Welcome to Anil Kumar's documentation space. You get a centralized hub to organize, manage, and collaborate on all your project documentation. Create pages, embed code snippets, and structure content intuitively without technical barriers. This space scales with your projects, from small notes to comprehensive guides.

## Quick Start

Get up and running in minutes.

<Steps>
  <Step title="Create Your Space" icon="plus">
    Sign up at `https://anil-kumar-docs.example.com` and start a new documentation space.
  </Step>
  <Step title="Add Your First Page" icon="file-plus">
    Click "New Page" and choose a template. Use the editor to add content.
  </Step>
  <Step title="Organize Content" icon="layers">
    Create folders and link pages. Preview your navigation sidebar.
  </Step>
  <Step title="Publish and Share" icon="share-2">
    Make it public or invite collaborators. Embed anywhere with a share link.
  </Step>
</Steps>

## Navigation Guide

Access content quickly using these methods.

<Tabs>
  <Tab title="Sidebar" icon="sidebar">
    Use the left sidebar for hierarchical navigation. Expand folders to browse pages.
  </Tab>
  <Tab title="Search" icon="search">
    Press `<kbd>Ctrl</kbd>+<kbd>K</kbd>` or click the search icon. Find pages by keyword instantly.
  </Tab>
  <Tab title="Breadcrumbs" icon="navigation-2">
    Track your location with breadcrumbs at the top. Jump back to parent pages.
  </Tab>
</Tabs>

## Customization Options

Tailor the space to match your brand. Apply the primary color `#3B82F6` via settings.

<CodeGroup tabs="CSS,JSON">
  ```css
  :root {
    --brand-primary: #3B82F6;
  }
  .header {
    background-color: var(--brand-primary);
  }
  ```
  ```json
  {
    "theme": {
      "primaryColor": "#3B82F6",
      "logoUrl": "https://your-logo.example.com/logo.png"
    }
  }
  ```
</CodeGroup>
